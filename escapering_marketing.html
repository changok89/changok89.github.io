탈출링 (EscapeRing)

가짜전화 시뮬레이션으로 자연스러운 상황 전환을 돕는 엔터테인먼트 앱입니다.

한국어 English 日本語

한국어

앱 소개

탈출링은 예약 또는 즉시 실행으로 착신 화면과 통화 연출을 재생하는 시뮬레이션 앱입니다. 실제 통화 서비스가 아닌 연출 목적의 앱입니다.

핵심 기능

  • 즉시 실행 / n분 후 / 특정 시간 예약
  • 착신자 이름, 번호, 테마 설정
  • TTS(한국어/영어/일본어) 및 문장별 음성 녹음
  • 자동 재생 / 탭 진행 방식
  • 백그라운드 알림 기반 실행

사용 흐름

시나리오 선택 또는 커스텀 작성 → 타이머 설정 → 착신 화면 시작 → 통화 연출 → 종료

English

Overview

EscapeRing is a fake-call simulation app that helps users trigger realistic incoming-call scenes on demand or on schedule. It is a simulation app and not a real calling service.

Key Features

  • Run now, run after N minutes, or schedule at a specific time
  • Customize caller name, number, and visual theme
  • TTS (Korean/English/Japanese) and per-line voice recording
  • Auto playback or tap-to-advance mode
  • Background execution with local notifications

Flow

Select or create a scenario → set timer → incoming screen → in-call simulation → end.

日本語

アプリ概要

EscapeRing は、即時または予約で着信演出を開始できる疑似通話シミュレーションアプリです。実際の通話サービスではありません。

主な機能

  • 今すぐ実行 / N分後 / 指定時刻で予約実行
  • 発信者名、番号、テーマのカスタマイズ
  • TTS（韓国語/英語/日本語）と文ごとの音声録音
  • 自動再生 / タップ進行モード
  • ローカル通知によるバックグラウンド実行

利用フロー

シナリオ選択または作成 → タイマー設定 → 着信画面 → 通話演出 → 終了
