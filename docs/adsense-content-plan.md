# AdSense content improvement plan

## Current diagnosis

The blog already has:
- domain/site available on GitHub Pages
- AdSense script in head
- Google site verification file
- robots.txt
- ads.txt
- About / Privacy pages

Main risk factors before re-review:
- many legacy posts from 2023 are very thin (roughly 65-250 words)
- several posts appear to be quick notes rather than complete tutorial/problem-solving articles
- site-wide quality can look inconsistent even if newer 2026 posts are much stronger

## Recommended strategy

### 1) Prioritize upgrading thin legacy posts

Upgrade these first because they are extremely short and topic-relevant:
- 2023-02-01 SwiftUI keyboard avoidance
- 2023-02-08 Mirroring
- 2023-02-07 iMazing
- 2023-02-02 thisisunsafe
- 2023-02-10 userAgent2
- 2023-02-07 Install_IPA
- 2023-02-05 lipo
- 2023-02-07 UDID
- 2023-02-08 Android USB Debugging

Target for each upgraded post:
- at least 700-1200 words
- explicit problem statement
- prerequisites / environment
- exact steps
- common failure cases
- verification steps
- conclusion / when to use the method

### 2) Add high-trust supporting pages
- Contact page
- Clear homepage intro explaining what the blog covers and who it is for
- Optional editorial policy / update policy page later

### 3) Publish 3-5 new strong technical posts before re-review
Suggested topic directions based on existing audience fit:
- Charles Proxy HTTPS inspection on Android 14 / recent devices
- Safari Web Inspector troubleshooting checklist for real devices
- GitHub Pages + Jekyll + AdSense setup notes from a developer perspective
- iOS Developer Mode troubleshooting on recent iOS versions
- OpenClaw / local AI automation practical setup with lessons learned

### 4) Re-review readiness checklist
- no obviously unfinished pages
- thin posts reduced or meaningfully expanded
- navigation includes About / Privacy / Contact
- at least several recent strong long-form posts visible on home page
- ads.txt served successfully at /ads.txt

## Fastest high-impact path
1. Add Contact page
2. Improve homepage intro
3. Expand 3 shortest legacy posts
4. Add 2 new detailed posts
5. Re-check live site and request AdSense review
