ClipDock

스트리밍 링크 감지부터 저장·재생까지, 한 앱에서.

ClipDock은 Android에서 m3u8/mp4 링크를 감지하고 내부 저장소로 다운로드한 뒤 재생/공유/이동/삭제까지 한 흐름으로 관리할 수 있는 스트리밍 워크스페이스입니다.

Android 10+ (SDK 29) | Last updated: 2026-03-09

한국어 English 日本語

Smart Detection

WebView 트래픽과 다운로드 이벤트를 기반으로 재생 가능한 m3u8/mp4 URL을 자동 감지합니다.

Reliable Pipeline

m3u8은 variant 해석 후 FFmpeg 기반 폴백 체인으로 처리하고, mp4는 직접 다운로드를 지원합니다.

Video Library

저장 영상 관리 화면에서 재생, 공유, 사용자 선택 경로 이동, 삭제를 빠르게 수행할 수 있습니다.

How It Works

  1. 원하는 페이지 URL을 열고 영상 재생을 시작합니다.
  2. ClipDock이 감지한 m3u8/mp4 링크 중 필요한 항목을 선택합니다.
  3. 다운로드를 실행하고 진행률을 확인하거나 필요 시 중지합니다.
  4. 저장 영상 관리 화면에서 재생/공유/이동/정리를 진행합니다.
